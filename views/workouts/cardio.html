Cardio Workout

Cardio workouts help build your fitness for long distance running and swimming. This cardio workout will help improve how long and hard you can run for:

  • 5 min warm-up run
  • 5 min active/dynamic stretches (dynamic stretching exercises)
  • 40-80 mins continuous run (75% effort)
  • Cool-down circuit (x3)
    • 10 burpees
    • 30 mountain climbers
    • 20 bunny hops
